## 📌 Overview

**Advance Medical Prescription Management Using Machine Learning** is an intelligent healthcare management system designed to digitize and automate prescription handling.

This platform helps **Doctors, Patients, Pharmacists, and Admins** manage healthcare operations efficiently through a secure and smart web application.

The system reduces paperwork, improves prescription accuracy, enables digital record management, and supports healthcare decision-making using Machine Learning.

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/Tanzilanaikwadi/Advance_Medical_Prescription_Management_Using_Machine_Learning.git
```

### Go to Project Folder

```bash
cd Advance_Medical_Prescription_Management_Using_Machine_Learning
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python app.py
```
